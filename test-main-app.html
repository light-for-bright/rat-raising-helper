Тест основного приложения

Инструкция: Откройте основное приложение по адресу http://localhost:8000 и перейдите во вкладку "Бланк учета". Проверьте, что расчеты работают правильно.
Проверка 1: Стартовый капитал
  1. Введите сбережения: 100
  2. Введите месячный денежный поток: 100
  3. Проверьте, что стартовый капитал = 200 ₽
Проверка 2: Текущий баланс
  1. Нажмите кнопку "Добавить"
  2. Введите сумму: 100
  3. Нажмите кнопку ➕ (доход)
  4. Проверьте, что текущий баланс = 300 ₽
Проверка 3: Расходы
  1. Нажмите кнопку "Добавить" еще раз
  2. Введите сумму: 50
  3. Нажмите кнопку ➖ (расход)
  4. Проверьте, что текущий баланс = 250 ₽
Проверка 4: Сохранение
  1. Обновите страницу (F5)
  2. Проверьте, что все данные сохранились
  3. Проверьте, что расчеты остались правильными

Ожидаемые результаты

Если все работает правильно:
✅ Стартовый капитал = 200 ₽ (100 + 100)
✅ Текущий баланс = 300 ₽ (200 + 100)
✅ Текущий баланс = 250 ₽ (200 + 100 - 50)
✅ Все данные сохраняются после обновления страницы

Если что-то не работает

Возможные проблемы:
• Стартовый капитал показывает 0 вместо 200
• Текущий баланс показывает неправильные значения
• Кнопка "Добавить" не работает
• Данные не сохраняются после обновления страницы
